# Linux File Permission Commands

## 📌 Commands Used

### 1. chmod
Changes file and directory permissions.

### 2. chown
Changes file ownership.

### 3. ls -l
Displays file permissions in detailed format.

### 4. chmod +x filename
Makes a file executable.

---

## 🎯 Learning Outcome
These commands help in managing access control and understanding Linux file permissions.
